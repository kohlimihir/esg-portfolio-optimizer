================================================================================
                    TROUBLESHOOTING
================================================================================

ERROR: "fatal: not a git repository"
FIX: Run "git init" first

ERROR: "remote origin already exists"
FIX: Run "git remote remove origin" then add again

ERROR: "failed to push some refs"
FIX: Run "git pull origin main --rebase" then "git push"

ERROR: "Permission denied (publickey)"
FIX: Set up SSH key or use HTTPS with personal access token

ERROR: Documentation files are showing up
FIX: Check .gitignore, run "git rm --cached filename", commit, push
