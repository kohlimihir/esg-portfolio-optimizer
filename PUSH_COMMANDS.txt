OPTION 1: USE THE BATCH FILE (EASIEST)

1. Initialize git (if needed)
2. Add all files
3. Commit with a message
4. Push to GitHub

================================================================================

OPTION 2: MANUAL COMMANDS (STEP BY STEP)
-----------------------------------------

# Step 1: Navigate to project
cd "C:\Users\asus\Downloads\Projects\ESG Model"

# Step 2: Initialize git (if not done)
git init

# Step 3: Add all files
git add .

# Step 4: Check what will be committed
git status

# Step 5: Commit
git commit -m "Initial commit: ESG Portfolio Optimizer with ML pipeline"

# Step 6: Create repository on GitHub
# Go to: https://github.com/new
# Name: esg-portfolio-optimizer
# Description: Production-ready ML system for ESG-based portfolio construction
# Click "Create repository"

# Step 7: Add remote (replace with YOUR URL)
git remote add origin https://github.com/YOURUSERNAME/esg-portfolio-optimizer.git

# Step 8: Push to GitHub
git branch -M main
git push -u origin main
